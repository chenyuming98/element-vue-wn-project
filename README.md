## 前端项目介绍



#### 技术介绍：elementUI+vue



#### 项目简介：第一阶段基础功能包含后台权限管理、公司人力管理、工作流；第二阶段整合电商项目的订单、商品上架、秒杀等业务操作。
















## 启动项目

```
#  安装依赖
npm install

# 运行
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```
